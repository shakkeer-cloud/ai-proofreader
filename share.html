⚙️ Supabase Configuration

One-time setup. Create a free project at supabase.com, run the SQL, then paste your credentials.

Step 1 — Run in Supabase SQL Editor
create table public.review_sessions ( id text primary key, report jsonb not null, doc_name text, initiated_by text, model_name text, created_at timestamptz default now(), expires_at timestamptz default (now() + interval '30 days'), status text default 'pending', reviewed_by text, reviewed_at timestamptz, decisions jsonb default '{}'::jsonb ); alter table public.review_sessions enable row level security; create policy "anon_read" on public.review_sessions for select using (true); create policy "anon_insert" on public.review_sessions for insert with check (true); create policy "anon_update" on public.review_sessions for update using (status = 'pending') with check (true);
Step 2 — Paste your credentials

Supabase → Settings → API → Project URL + anon public key.
